# E-commerce-Product-Page
A product page for an e-commerce website.
This project includes:
  - Product Images
  - Product Descriptions
  - Product Prices
  - Shopping Cart Functionality (Add to Cart, View Cart,Remove from cart)

Languages used: HTML,CSS,Javascript
Also used: JSON, Bootstrap

index.html is the main page and catalogue displays different types of products available.
Select the type of product required and explore the options.
You can add the product to cart and view cart by clicking on the cart button.
In the cart, you can also remove the product and pay for the remaining products...
